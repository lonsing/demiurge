string StringUtils::replaceAll ( const string &  in,
const string &  from,
const string &  to 
)
static

Returns a copy of a string where all occurrences of a certain pattern are replaced.

This method copies the passed string, replaces all occurrences of the string 'from' with the string 'to', and returns the resulting string. That is, StringUtils.replaceAll("Evil devil", "vi", "mai") returns "Email demail". If 'from' is the empty string, nothing is replaced.

The only difference to replaceAllIn is that this method does not modify the passed string.

Parameters
inThe string in which the replacement should be made.
fromThe string from which all occurrences should be replaced in 'in'.
toThe string with which all occurrences of 'from' should be replaced.
Returns
The resulting string where the replacement has been performed.
