A macro for an assertion that is only enabled in debug mode.

In debug mode, if the condition evaluates to false, the error message is printed and the program terminates. If not in debug mode, the statement is ignored (the condition is not even evaluated).

Parameters

messageThe error message to be printed upon assertion violation.
