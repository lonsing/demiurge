Detailed Description

A generic interface to QBF-solvers started as external processes.

This is an abstract base class for all interfaces to QBF-solvers which start the QBF-solver in a separate process. Practically all QBF-solvers understand the QDIMACS format as input. Hence, different QBF-solvers can be interfaced in the same way. The only things that differ from solver to solver are the location of the executable and the command-line options that must be set. These solver-specific aspects are handled in the derived classes (by overriding certain methods). Everything else is handled in this abstract base class. At he moment there are two implementations of this abstract class: DepQBFExt and QuBEExt.

)

Checks if a given QBF is satisfiable (with quantification over variable kinds).

The QBF consists of a quantifier prefix and a Boolean formula in CNF. The quantifier prefix assigns an existential or universal quantifier to every kind of variable. The QBF is dumped into a file, a QBF-solver is called, and the exit code is used to infer the satisfiability result.

Parameters
quantifier_prefixThe quantifier prefix as a vector of pairs. vector[0] is the leftmost (i.e., outermost) quantifier block. The pairs in the quantifier_prefix assign an existential or universal quantifier to every kind of variable that occurs in the cnf.
cnfA Boolean formula in CNF.
Returns
True if the QBF is satisfiable, false otherwise.

bool ExtQBFSolver::isSatModel ( const vector< pair< VarInfo::VarKind, Quant > > &  quantifier_prefix,
const CNF &  cnf,
vector< int > &  model 
)
virtual

Checks satisfiability and extracts a model (quantifying over variable kinds).

Just like isSat() , this method checks a quantified Boolean formula for satisfiability. If the QBF is satisfiable, this method also extracts a model (a satisfying assignment) for all variables which are quantified existentially on the outermost level. The model is provided as cube: negated variables in the cube are FALSE, unnegated ones are TRUE.

Parameters
quantifier_prefixThe quantifier prefix as a vector of pairs. vector[0] is the leftmost (i.e., outermost) quantifier block. The pairs in the quantifier_prefix assign an existential or universal quantifier to every kind of variable that occurs in the cnf.
cnfA Boolean formula in CNF.
modelThe resulting model in form of a cube in case of satisfiability. Negated variables in the cube are FALSE, unnegated ones are TRUE. If the formula is unsatisfiable (this method returns false) then this parameter is not modified.
Returns
True if the QBF is satisfiable, false otherwise.
