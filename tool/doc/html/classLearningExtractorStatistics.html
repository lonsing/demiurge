Detailed Description

A class to collect performance measures for the LearningImplExtractor.

Author
Robert Koenighofer (rober.nosp@m.t.ko.nosp@m.enigh.nosp@m.ofer.nosp@m.@iaik.nosp@m..tug.nosp@m.raz.a.nosp@m.t)
Version
1.2.0

This class collects statistics and performance data for the learning-based synthesis circuit extraction method implemented using QBF or SAT solving (implemented in the class LearningImplExtractor).
