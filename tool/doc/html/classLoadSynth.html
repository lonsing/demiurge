Detailed Description

A dummy back-end which loads the winning region from a file.

This back-end is used to reduce the computation time when benchmarking relation determinization approaches. It loads the winning region from a file instead of computing it. The StoreImplExtractor can save the winning region into a file. We assume that this has been done before.
