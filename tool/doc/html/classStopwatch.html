Detailed Description

Implements a stopwatch that can be used to measure execution time.

The stopwatch is able to find out both elapsed CPU time and elapsed realtime. CPU time is meant to be used for precise measurements of short time periods. For longer periods, it may overflow (typically after 36 minutes in a 32 bit machine; on a 64-bit machine this usually does not happen). Use getRealTimeSec() to crosscheck.

Usage example:

PointInTime start_time = Stopwatch::start();
doFirstPart();
double first_part_cpu = Stopwatch::getCPUTimeSec(start_time);
L_LOG("doFirstPart() took " << first_part_cpu << " seconds of CPU time.");
doSecondPart();
double all_real_time = Stopwatch::getRealTimeSec(start_time);
L_LOG("The whole computation took " << all_real_time << " seconds (real-time).");
